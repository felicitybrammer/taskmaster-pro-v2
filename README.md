# TaskMaster Pro (v2)

## Description

A Kanban-style taskmanagement application that features drag-and-drop list items, deadline management via colour-coded warning labels, and updated UI interface. 

## Built With:

* jQuery
* Bootstrap
* jQuery UI
* DayJS
